# Modal verbs

Modal verbs are verbs like can, must and like. They show possibility, ability,
intent, obligation or necessity. Most of the time they come in combination with another verb.

## Können

Können is an irregular verb.

|               |        |
|---------------|--------|
| **ich**       | kann   |
| **du**        | kannst |
| **er/sie/es** | kann   |
| **wir**       | können |
| **ihr**       | könnt  |
| **Sie/sie**   | können |

## Müssen

| **du**        | musst  |
| **er/sie/es** | muss   |
| **wir**       | müssen |
| **ihr**       | müsst  |
| **Sie/sie**   | müssen |
